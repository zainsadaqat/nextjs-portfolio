# My Personal Portfolio

## Built With

- NextJS, Styled Components

## Steps to run the project in your local machine

Go To Terminal and paste the following command:

`git clone git@github.com:zainsadaqat/nextjs-portfolio.git`

`cd nextjs-portfolio`

`npm install`

`npm start`
